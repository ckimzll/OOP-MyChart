# 📊 나만의 차트만들기
객체지향프로그래밍 수업 "나만의 차트만들기" 과제

### 주요 기능 
- 5학기까지 수강한 강의를 전공, 교양으로 나누어 그동안 채운 학점과 더 수강해야 하는 학점을 원형차트, 선형차트, 막대차트로 분석
- 1~4학기까지의 학점을 목표학점인 3.7이상이면 눈이 내리고 미만이면 비가 내리는 눈사람차트
- 5학기 성적을 이후에 입력하면 최종학점을 계산해주는 차트
